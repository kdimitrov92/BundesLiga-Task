Interview task for the position of Senior Python Dev at UpdateOne

### Prerequisites:
* Python 3.11
* PIP
___________________________________________________

### Instructions for running the project:

* Install all prerequisites for this repository
* run `pip install requirements.txt` for the project requirements
* Use runserver to start the Django server access it on the default localhost:port url

 (Usually that would be `http://127.0.0.1:8000/`)
